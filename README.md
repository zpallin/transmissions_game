# Rainy Days
Explore the city, find your purpose and transform the city! 

Built at Global Game Jam 2018.

## Technologies
- Express: 4.16.2,
- Pixi-Particles: 2.1.9,
- Pug: 2.0.0-rc.4 

## How to Run This Repo
1. ```Git clone``` the repo
2. Run ```npm install```
3. Run ```npm start```
